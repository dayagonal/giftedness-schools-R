# giftedness-schools-R

## Описание проекта

Цель проекта — выявить факторы, связанные с академическими достижениями учащихся на уровне школ.

## Данные

В работе использовались внутренние датасеты, включающие:

- Результаты олимпиад учащихся
- Информацию о школах
- Данные о педагогах и их категориях
- Региональные и инфраструктурные характеристики школ

**Исходные данные не публикуются из-за их чувствительности**

##  Моделирование

В работе использованы следующие подходы:

### 1. Логистическая регрессия
- Зависимая переменная: одарённость (бинарная)
- Независимые переменные:
  - инфраструктура школы
  - качество педагогов
  - тип местности (город/село)
  - сменность обучения

### 2. Линейные регрессии
- Анализ влияния факторов на успеваемость по предметам
- Используются перцентильные значения результатов

### 3. Дополнительные тесты
- t-тесты для сравнения городских и сельских школ
- проверка гипотез о влиянии территориального признака
